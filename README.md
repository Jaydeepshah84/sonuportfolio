# Sonu Singh — UI/UX & Product Designer Portfolio

A premium, animated single-page portfolio built with **React + Vite**, **Framer Motion**, **GSAP (ScrollTrigger)** and **Lenis** smooth scrolling.

## ✨ Features
- Animated hero with screenshot background montage, 3D photo tilt & floating Figma badge
- Scroll-reveal sections, magnetic buttons, animated counters & skill bars
- Real project imagery, design-process timeline, photo testimonials
- Smooth scrolling (Lenis) synced with GSAP ScrollTrigger
- Fully responsive · dark glassmorphism theme

## 🛠 Tech
React 18 · Vite 5 · Framer Motion · GSAP · Lenis

## 🚀 Local development
```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build → dist/
npm run preview  # preview the build
```

## ▲ Deploy on Vercel
1. Push this repo to GitHub.
2. On [vercel.com](https://vercel.com) → **Add New → Project** → import this repo.
3. Vercel auto-detects **Vite** — Build: `npm run build`, Output: `dist`.
4. Click **Deploy**.
